# KOOMPI OS Desktop Configurations
# This directory will contain pre-configured dotfiles for KOOMPI desktop editions.
# 
# Structure:
#   hyprland/     - Hyprland + Waybar configs (KOOMPI Tiling)
#   openbox/      - Openbox + tint2 configs (KOOMPI Lite)
#   foot/         - Terminal emulator config
#   fuzzel/       - App launcher config
#   zellij/       - Terminal multiplexer config
#   fastfetch/    - System info config
#
# These configs are applied by koompi-desktop during DE installation.
