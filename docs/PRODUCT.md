# Product

## Register

product

## Users

KOOMPI OS users: students and teachers in Cambodian schools, developers, founders, and privacy-conscious users worldwide. The brainstorm page itself is read by contributors, designers, and stakeholders evaluating the OS vision; the desktop simulations inside it depict the actual OS shell (Quickshell on Hyprland) that end users will live in daily, on modest hardware (Intel UHD 620 class).

## Product Purpose

KOOMPI OS is "The OS That Thinks": an AI-first, local-first Linux OS where the index, models, and memory stay on-device by default. The deeper thesis: KOOMPI turns the computer from a tool into a partner — it thinks, remembers, evolves, and grows with its user across decades. It is also Web3-native: self-sovereign keys, identity, and payments are OS primitives (Selendra-default), consent-first and never chain-gated. The brainstorm page's desktop sims exist to make the shell vision tangible: a reader should feel what the real OS will feel like — calm, precise, private, alive. Success: a stakeholder clicks through a sim and believes the OS is real and premium.

## Brand Personality

Calm, sovereign, precise. The interface is quiet and confident, never loud; intelligence is ambient (indigo), data is visible (cyan), safety is explicit (emerald). Khmer-first dignity: formality and craft over flash.

## Anti-references

- Surveillance-era cloud OS aesthetics (Chrome OS marketing gloss, telemetry-cheerful)
- Generic SaaS dashboard styling (hero metrics, identical card grids)
- Gamer-RGB / neon-cyberpunk "AI" cliché
- Windows 95 nostalgia or deliberately retro hacker UI

## Design Principles

1. Privacy is a visible state, not a setting: color tokens are contracts (emerald = local/safe, amber = consent, red = violation, indigo = AI, cyan = data).
2. Awareness without intrusion: the OS senses much, shows everything, interrupts almost never.
3. Honest motion: animation conveys state (opening, thinking, loading), never decoration for its own sake.
4. Runs beautifully on modest hardware: effects degrade gracefully; nothing depends on a flagship GPU.
5. A 50-year relationship: calm enough to live in for decades, not a launch-week impression.

## Accessibility & Inclusion

A11y is a build gate in the real OS (AT-SPI roles required in CI). For the brainstorm page: WCAG AA contrast, `prefers-reduced-motion` respected everywhere, keyboard reachable where interactive, Khmer text rendered with correct segmentation.
