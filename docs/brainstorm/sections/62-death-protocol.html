Death Protocol & Digital Estate

Futures 1.x

Inheritance is the legacy preset of the shared Lifeboat export engine (§24) plus a documented estate handoff. At 1.x you map each memory domain to an heir, split the executor key among people you chose, and gate release behind a dead-man timer and executor quorum. Unbequeathed per-object keys are shredded with the master key — no vendor holds a copy, so there is nothing to subpoena. This is a key-rewrap and export flow, not a "digital afterlife".
