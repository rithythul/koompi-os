Ambient Intelligence

AI 1.x

A Context Pulse widget that surfaces the signals koompi-contextd already emits — nothing more. At 1.x it shows your next calendar event, uncommitted git changes, a focus/pomodoro timer, and the active window. Every value is read from a local contextd event; the widget renders state, it never acts on it. If the Pulse doesn't display a signal, contextd isn't reading it.
