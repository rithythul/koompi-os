What KOOMPI Holds On-Device
Context Index
Semantic embeddings of your files. Stored in ~/.local/share/koompi/context.db. Never leaves device.
AI Memory
Conversation summaries and named facts from AI sessions. Encrypted with your login key.
Usage Patterns
App launch frequency, active window durations, focus sessions. Used locally for proactive suggestions only.
Location (opt-in)
City-level location for weather. Stored as lat/lon with 2-decimal precision (~10 km). Never transmitted.
Health (future)
Eye strain, posture reminders, focus scores from wearables. On-device only, no cloud health sync.
Selendra Identity
DID keypair and verifiable credential proofs. Private key never leaves the device TPM.
You own all local data KOOMPI

All data stores are in standard formats under ~/.local/share/koompi/. Export or delete any category at any time from this dashboard — no vendor lock-in.

Selendra ID: proof without disclosure

Your digital identity uses zero-knowledge proofs via Selendra blockchain. You can prove "I am over 18" or "I own this key" without revealing your name or address.

Consent receipts

Every outbound data flow requires explicit user consent. Receipts are stored in the Egress Ledger with timestamp, destination, data category, and your consent action.
