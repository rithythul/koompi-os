Draggable canvas

Widgets are free-floating. Drag by the header bar. Resize from any edge. Position persists across sessions per workspace.

AI Quick Ask widget KOOMPI

Inline AI chat box that stays on screen. Ask about current context without opening the full sidebar. Results appear inline in the widget.

Persistence

Widget layout saved per workspace. Desktop 1 can have resource monitor; Desktop 2 can have notes. Super+G toggles the overlay canvas without disturbing running apps.

KOOMPI widgets vs end-4

end-4's ags has a widget system; KOOMPI adds the AI Quick Ask as a first-party widget. All other widgets (notes, monitor, timer) follow the same drag/resize API.
