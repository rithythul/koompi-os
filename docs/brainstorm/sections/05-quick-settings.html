Privacy Mode tile KOOMPI

Local Mode tile uses emerald "priv-on" state to distinguish it from normal on/off toggles. Tapping it opens the Privacy Dashboard for granular control.

AI toggle KOOMPI

Disabling AI pauses all background inference, RAG indexing, and context collection. The bar pill dims. Useful for battery saving or focus mode.

Slider interactions

Brightness and volume sliders are interactive. Dragging updates the OSD overlay in real time. Scroll wheel on either slider also works from keyboard.
