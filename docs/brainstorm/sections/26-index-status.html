100% local processing Privacy

All embedding and vector generation runs on-device via a quantized local model. No file content is transmitted externally during indexing.

Incremental watch via inotify

koompi-contextd subscribes to kernel inotify events. Only modified or new files are re-indexed — full reindex only on schema upgrades or user request.

Semantic + keyword dual index

Each file produces both a vector embedding (semantic search) and a keyword token index (exact match). The AI sidebar queries both and merges results by relevance score.
