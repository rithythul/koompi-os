Agent Memory Blueprint

AI

The implementation design for the memory engine inside koompi-assistantd (Rust, per ADR-0004 — memory shares the one per-user SQLite DB with L1, arch §7). The Rust core owns the durable substrate and IPC; a NeuroAgentMemory spec encodes the four-layer cognitive logic. Two invariants govern everything: the durability split (forget the index, never the record) and embedding-model obsolescence (raw episodic text is immortal so the index can be re-embedded under future models). Forgetting is a ranking property; remembering is reconstruction with citations.

The Rust core is the only code path that touches disk, sockets, and the encryption key. The Python class below is the executable specification of the cognitive logic — the shipping implementation is Rust inside assistantd (ADR-0004); Python stays on dev machines for spec and evals, never resident on user hardware. Every vector carries an embed_model_id so a 2050 model never silently re-interprets who you were at 25.

spec/neuro_agent_memory.py · ships as Rust in assistantd executable spec · LOCAL-FIRST
# Durability split: episodic = immortal source of truth; index = rebuildable. # Provenance: every vector carries embed_model_id → era-correct reconstruction. class NeuroAgentMemory: def __init__(self, store, embedder, local_llm, ctx_window_tokens: int): """Initialise all four cognitive layers over ONE embedded store.""" self.store = store # sqlite-vec + FTS5 + graph (assistantd-owned) self.embedder = embedder # local ONNX; .model_id is the tag self.llm = local_llm # LOCAL ONLY; never cloud for episodic self.working = [] # ① PFC: in-RAM attention ring buffer self.budget = ctx_window_tokens# slot sum capped to model window − reserve # ② Episodic, ③ Semantic, ④ Procedural live as namespaces in `store`. def perceive(self, event_text: str, context: dict, tokens: int, novelty: float): """Route a perceived event into working memory; evict under pressure.""" slot = {"text": event_text, "ctx": context, "tokens": tokens, "salience": novelty + context.get("arousal", 0.0), "touch": time.time()} while sum(s["tokens"] for s in self.working) + tokens > self.budget: victim = min(self.working, key=lambda s: s["salience"] * self._recency(s["touch"])) self.working.remove(victim) self._flush_episodic(victim) # evicted ≠ lost: it happened self._potentiate_coactivation(victim) # it still shaped the session self.working.append(slot) async def consolidate_sleep_cycle(self): """S2 deep sleep: local-LLM distils episodic logs → semantic graph. Append-and-reconcile; contradictions timestamped, never overwritten. Returns early (defers) if no AC power / idle / capable local LLM — LOCAL-FIRST is a control-flow guarantee, never a cloud fallback.""" if not (self._on_ac_power() and self._cpu_idle() and self.llm.is_capable): return # DETERMINISTIC DEGRADE — defer window = self.store.episodic_since(self.store.last_consolidation_ts()) for batch in self._batched(window): data = json.loads(await self.llm.complete(CONSOLIDATION_PROMPT, batch)) for e in data["entities"]: self._upsert_node(e, refs=[r.id for r in batch]) for r in data["relations"]: self._merge_relation(r) # Hebbian bump / reconcile self._propagate_salience() # PageRank pass self.forget(threshold=PRUNE_θ) # index-only decay sweep self.store.mark_consolidated(time.time()) async def reconstruct_memory(self, current_context: dict) -> dict: """Constructive recall: blend query across layers, hybrid-retrieve, graph-expand by association, rerank, narrate with provenance. Recall strengthens stability (testing effect). Spans tagged recorded|inferred — TRANSPARENT AI, never silent confabulation.""" qv = self._blend_query( goal=self.embedder.embed(current_context["goal"]), valence_bias=current_context.get("mood", 0.0), temporal=current_context.get("era"), active_set=self.working) cands = self.store.hybrid_retrieve(qv, k=64) # vec KNN ⊕ FTS5 (RRF) cands = self.store.graph_expand(cands, depth=2, min_weight=EXPAND_θ) ranked = sorted(cands, key=lambda n: self._rank(n, current_context), reverse=True)[:12] for n in ranked: # testing effect n.access_count += 1; n.last_access = time.time(); n.stability *= 1.05 sources = self.store.episodic_for(ranked) narrative = await self.llm.complete(RECONSTRUCT_PROMPT, nodes=ranked, sources=sources) return {"narrative": narrative, "sources": [s.id for s in sources], "reconstructed": True, "embed_eras": sorted({n.embed_model_id for n in ranked})} def forget(self, threshold: float): """Ebbinghaus decay → prune INDEX rows only. @data is untouched. A pruned memory is one the OS won't surface unprompted — but the episodic_refs back-pointers keep it recoverable from @data forever.""" now = time.time() for n in self.store.all_nodes(): dt = now - n.last_access R = math.exp(-dt / (n.stability * (1 + 0.3 * n.access_count))) rank = n.salience * math.log1p(n.access_count) * R if rank < threshold and self.store.has_near_duplicate(n): self.store.prune_index_row(n) # collapse → representative # n.episodic_refs stay valid → immortal in @data

reconstruct_memory · scoring formula constructive, not top-k
# ── Query blend (mood-/goal-/era-congruent recall) ────────────── query_vector = w_g·goal_emb + w_e·valence_bias + w_t·temporal_anchor + w_a·active_set w_g=.50 w_e=.15 w_t=.20 w_a=.15 # ── Ebbinghaus retention (per node) ───────────────────────────── S(n) = S₀·(1 + β·access_count) # stability grows w/ recall β=0.3 R(n) = e^(−Δt / S(n)) # forgetting curve # ── Final retrieval rank ──────────────────────────────────────── rank(n) = salience(n) · log(1+access_count) · R(n) · goal_align(n) └ PageRank └ testing effect └ decay └ cos(n.vec, query_vector) # salience seeded by intrinsic importance (amygdala-modulated): salience(n) = (1−d)·base(n) + d·Σ_{m→n}[ w(m,n)/Σw · salience(m) ] d=0.85 base(n) = emotional_intensity(n) · novelty(n)
Privacy Guarantees
✓ All processing local. Embedding + LLM + graph run on-device. Episodic data never egresses for consolidation — S2/S3 defer until a capable local session exists.
✓ Encryption at rest. Per-object AEAD via koompi-vault (O-1); key in the per-user keyring, TPM-sealed where hardware exists. Honest residual: a session-unlocked keyring is readable by same-UID code (arch §9).
✓ User audit trail. AuditTrail() exposes every read/write/inference the engine made — who asked, what surfaced, which sources cited.
✓ Sovereignty. Sync is opt-in E2EE; index decay never touches episodic; --full (hand-off) wipes both for the leaving user — destruction is real when asked (§30, §73); Lifeboat exports episodic at will.

The durability split invariant

Decay, pruning, and the Ebbinghaus curve govern the derived index — edge weights, salience, working-memory eviction. They never touch raw episodic records. Forgetting is a retrieval property, not a storage operation: a pruned memory rebuilds from the immortal @data substrate given the right cue.

Re-embeddable across decades 2050-proof

BGE-M3 will not exist in 2050 — its first credible successor appeared within a year of the pin (§87, G-EMB-2). Because raw text is immortal and every vector carries embed_model_id, the index re-embeds under future models via a runbook exercised in CI — and "who you were at 25" is reconstructed from era-correct vectors, never silently re-framed by a 2050 lens.

Recorded vs inferred, always

Every reconstructed span is tagged recorded (an episodic row, clickable + timestamped) or inferred (LLM reconstruction, flagged). An OS entrusted with a human life that confabulates is a betrayal; this one is auditable down to the timestamped row.
