Agents exchange offers and hard limits, never the private context behind them. "Thursday is blocked" crosses the mesh; why never does. The Household Constitution is the treaty each agent carries into every room.

Twin-ledger accountability

Every negotiation writes matching, hash-linked receipts into both owners' Egress Ledgers through koompi-egress. A disagreement about what was shared has a ground truth on both machines.

A society needs an open protocol

Agent-to-agent negotiation is the App-Context Bus extended over the mesh — MCP-shaped, openly specified. If only KOOMPI agents can sit at the table, it isn't diplomacy; it's a walled garden with better manners.
