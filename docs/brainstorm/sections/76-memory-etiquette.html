Etiquette cannot require the other person to own a KOOMPI. The QR works from any phone; the spoken phrase works from any mouth. Manners degrade gracefully.

Conversations have two owners KOOMPI

Every memory platform treats whatever the user hears as the user's data. KOOMPI is the first to model the other party's stake: remembering-preferences are exchanged machine-to-machine and enforced by koompi-agent-memd at write time, not filtered after the fact.

Honored, and provably so

An etiquette promise without an audit trail is a vibe. Receipts land in both Egress Ledgers, so Vathna's OS can verify Rithy's kept its word — the same single-gate accountability that governs network egress, turned toward social obligations.

Etiquette is culture, shipped as protocol

What a machine owes a guest is a deeply Cambodian question — hospitality has rules. Encoding them as a mesh protocol and a notice by the door makes KOOMPI's politeness exportable: the first OS whose defaults include respect for people who never touch it.
