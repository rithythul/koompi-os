A notification makes you do the work. A proposed action does the work and asks for a yes. The OS arrives at the start line already prepared.

The consent boundary is bright

Reading, drafting, and preparing are free. Anything irreversible — sending, posting, paying, committing — always stops at an Allow / Edit / Dismiss tap. The line never moves silently.

Always explainable

Every proactive action exposes its reasoning chain: the trigger, the inference, the gap it spotted, and the data it touched. No black-box helpfulness.
