🔒 Memory stored locally · never sent to cloud
Local-only storage KOOMPI

All memory entries are stored in ~/.local/share/koompi-ai/memory.json. Never synced to a cloud service. Local model reads them directly.

User control

Each entry has a × delete button. "Clear all" wipes all memory with a confirmation step. Individual entries are also editable — click text to edit inline.

AI uses this for personalization

On each conversation start, relevant memory entries are injected into the system prompt. User type entries = who you are. Project type = current context. Feedback type = AI behavior preferences.
