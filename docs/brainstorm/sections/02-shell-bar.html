Shell Bar

Quickshell / QML

Three bar styles, each a fully interactive desktop simulation — click any element and panels animate open just like the real shell. Hug (default) — top bar flush to screen edge with concave fillet corners. Taskbar — Windows 11 inspired: bottom full-width bar, centered launcher, Mica translucency, notification bell. Dock — macOS inspired: thin top menu bar + floating dock pill with hover magnification. All styles show KOOMPI's Privacy + AI identity pills.

DockClick the menu-bar K (start), AI pill or ✦ dock icon (control center), 🔍/▦ dock icons (launcher/overview), clock (calendar), brightness & volume. Click a window or dock icon to focus. Titlebar ─ ▢ ✕ are live; dock icons restore. Esc closes any panel.
Hug (default) — bar flush to screen edge. Concave fillet "ears" at bottom corners match the display bezel radius. Glass spans full width. Most immersive.
Workspaces pill Center

Active=indigo, occupied=dim, empty=ghost. Click workspaces = Overview. Max 10 shown.

Pomodoro pill Center-left

Circular arc + phase icon + MM:SS countdown. Click = popup. Dim when idle. Work phase = red, break = cyan.

Privacy pill KOOMPI

Always visible. LOCAL=emerald, CLOUD=amber, OFF=dim. Pulses to signal active egress gate. Tapping opens Privacy Dashboard.

AI pill KOOMPI

✦ idle · spinner thinking · ✓ done. Click = toggle AI sidebar. Right-click = new conversation.

Left side scroll

Scroll over left zone = brightness. Hover reveals brightness hint icon + tooltip.

Right side scroll

Scroll over right zone = volume. OSD appears. Hover reveals volume hint.

Panel Style — Design Notes Win11 + macOS inspiration
Dock pill Panel

Centered capsule with K Start button + pinned apps. Running apps get a dot indicator below. Separator divides pinned from utility apps. macOS-style magnification on hover is optional.

2-line clock Panel

Time above, date below (Win11 signature). Stacked in 12px/10px. Clicking opens Calendar popup. Right side keeps tray cluster grouped — network + volume + battery in one pill.

Mica translucency Panel

Higher saturate() blur vs top-bar styles. Wallpaper color bleeds through more visibly — bar feels anchored to the desktop, not floating above it. Dark: rgba(8,14,26,0.82).

Open question

Panel has no Pomodoro or Workspaces pill visible. Options: (1) move workspace dots below dock icons as a row, (2) workspace dots in top-right corner, (3) omit — use Overview gesture instead.

Dock — macOS Style Notes menu bar + floating dock
Menu bar Dock

Active app name + global menus on left. Status cluster on right. 24px thin strip.

Icon size Dock

44px (vs 34px for Taskbar). Larger touch targets, more visual weight.

Magnification Dock

scale(1.28) + translateY(-6px) on hover — the classic macOS spring.

Open question

Does the KOOMPI AI icon belong in the dock, or in the menu bar only?

Open question

Dock autohide — slide down on idle, reveal on mouse approach bottom edge.

Bar Popup Widgets

Every bar cluster opens a contextual popup. Popups are layered QML panels anchored to the bar element — same glass + blur treatment as the sidebar.

Media (MPRIS) Right zone

Visible only when media is active. Shows album art thumbnail + truncated track name + prev/play/next controls. Max-width shrinks on narrow screens. Click = full popup. Scroll on pill = volume.

Util buttons Right zone

Compact icon trio: Screenshot (click = region, shift-click = fullscreen), Color picker (hyprpicker, copies hex to clipboard), Screen recorder (wf-recorder; turns red when recording).

SysTray Right zone

SNI protocol. App icons auto-populated at runtime — Discord, Telegram, etc. Click = activate/hide window. Right-click = app tray menu. Hidden overflow → expand button appears when tray overflows.

Display management KOOMPI

Brightness scroll on left zone (existing). Display icon in util buttons opens the popup: brightness + night light + refresh rate + scale + multi-monitor arranger. No separate Settings app needed for daily adjustments.
