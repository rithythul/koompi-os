The stamps six words that turn a brainstorm into a register
v1
Ships with Naga. Capability-gated, never event-gated — there is no launch date, only readiness.
1.x
Committed; lands after v1 in track order. A promise with a queue position, not a date.
GATED(id)
Ships when the named gate passes its measurable threshold — never on a date.
RULE
Not a feature: an engineering rule enforced from now, by ADR or CI.
HORIZON
Vision; lives on the shelf. Only its named kernel is engineering.
CUT
Removed, reason recorded. Re-opening requires new dated evidence — never nostalgia.
A stamp may carry a delta — the part of the brainstorm section that does not survive into the commitment, referencing the corrections register below. One special stamp, ADOPT (§45), means: use the ecosystem's tool, build nothing.
The 2026-06 calibration five verified facts set the posture of every stamp — sources in §87
The RAM crisis
DDR5 ≈3–4× year-over-year; relief no earlier than late 2027. T0/T1 hardware is the mainstream product through 2028.
Small models target tools
1–4B models ship with function calling by design; grammar-constrained decoding makes malformed calls structurally impossible. Local agency at the Floor is real — bounded.
Khmer speech flipped
106.5 h curated ASR corpus; fine-tuned Whisper ≈8% CER; open TTS recipe + Piper runtime. Voice moves from research to gated 1.x. Dialects stay out.
MCP won the tool layer
Linux Foundation 2025-12, >10k servers. L3 adopts the protocol and invents only the leash (ADR-0013).
The window is open
Recall shipped and is distrusted; Copilot+ gates AI behind 40 TOPS + 16 GB. The same memory, provably yours, on hardware you already own — ≈2026–2028.
