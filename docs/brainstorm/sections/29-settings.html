Settings as QML app Native

Settings is a first-party QML application rendered via Quickshell. Same design system as the shell — consistent typography, spacing, and animations across all panels.

Privacy settings are top-level

Privacy, AI, and data controls are surfaced at the same level as appearance — not buried under "Advanced". The assumption is these matter to every KOOMPI user.

Deep-link from notifications

Any system notification can carry a settings deep-link. "An app wants network access" opens directly to Egress Rules for that app. No path memorization required.
