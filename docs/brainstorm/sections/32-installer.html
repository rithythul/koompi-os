Zig-native TUI renderer Zig

The installer TUI is a single statically-linked Zig binary. No Python, no bash dependency chains. Boots in under 100ms and runs on any terminal with VT100 support.

AI setup is step 4, not optional

AI configuration is a first-class install step — not buried in settings post-install. The local model is selected, downloaded, and tested before the user ever reaches the desktop.

Selendra ID at install time

Users can generate or import a decentralized identity during setup. This anchors their KOOMPI profile to a portable, verifiable identity they can carry across devices.
