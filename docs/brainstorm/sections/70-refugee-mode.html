Verifiably nothing to find

Border-crossing mode is not hiding — it is genuine absence. koompi-restore crypto-erases the device and can attest the factory state. An inspector examining the machine finds the truth: there is nothing on it.

Built where the history demands it

An OS designed in Cambodia carries the memory of what happens when people must flee with nothing. Refugee mode is that memory turned into engineering — quiet, factual, and ready before it is ever needed.
