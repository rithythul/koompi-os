Khmer-Native Intelligence

i18n v1

v1 ships a fully Khmer OS surface: register-correct UI strings, honorifics, and the Khmer + Buddhist calendars, authored and reviewed so the shell reads as native rather than machine-translated. This is a strings-and-locale deliverable, not a language model. Khmer chat — an assistant that reasons and converses in Khmer — is a separate track gated behind G-KM-CHAT-1: it ships only after a Sailor2 / SEA-LION-class local model passes the Khmer eval. Until that gate clears, KOOMPI does not present fluent Khmer conversational AI.

Language independence is data independence KOOMPI

A Khmer-first surface means 17 million people compute in their own language locally, without shipping keystrokes to a foreign cloud to be understood. That holds at v1 for strings; it holds for chat once G-KM-CHAT-1 clears.
