Register-aware AI — unique to KOOMPI Khmer

Khmer has three distinct registers: informal (ធម្មតា), formal (គោរព), and royal (ព្រះរាជ). The AI uses the correct pronouns, verbs, and honorifics for each — not a simple translation layer, but grammatically correct Khmer.

Noto Khmer + custom hinting

The shell uses Noto Khmer with custom subpixel hinting tuned for HiDPI displays common in KOOMPI hardware. Subscripts and stacked consonant clusters render without the baseline artifacts common in other desktop environments.

Locale-aware number and date formatting

Khmer numerals (០–៩), Buddhist calendar dates, and currency (Riel ៛) are formatted natively throughout the OS — not just in the locale settings panel. Bar clock, file timestamps, and settings all respect the active locale.
