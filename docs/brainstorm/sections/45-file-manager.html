Browsing and asking are one bar KOOMPI

macOS keeps Finder and Spotlight separate; GNOME hides search behind a keystroke. KOOMPI fuses them: the address bar parses a path, or a sentence in Khmer or English. The ✦ ASK MODE pill tells you which interpretation contextd chose — and you can force either. No second window, no context switch.

Privacy state per file, always visible

Every row shows what the system knows about it: indexed (data the AI can read), encrypted (at rest), shared (left this device with consent), and leaked-once (left once via an app — auditable in the egress log, revocable). The file system is legible, not a black box the AI quietly reads.

Spatial memory beats timestamps

Finder's "Recents" sorts by mtime — useless when you remember context, not time. KOOMPI records where you last saw a file: attached to an email, previewed in Telegram, opened beside a calendar event. Over a lifetime, this is how humans actually find things — by the situation, not the clock.

Khmer is sorted correctly, not byte-wise

Naive strcmp on Khmer produces nonsense dictionary order — dependent vowels and the coeng subscript (U+17D2) reorder under the Unicode Collation Algorithm's Khmer tailoring. And because Khmer has no inter-word spaces, content search tokenizes through ICU's dictionary break-iterator, not whitespace splitting. A Khmer user's files sort and search exactly as their language demands — first-class, never an afterthought.

Honest about partial knowledge

While a folder indexes, results say so — "117 of 191 embedded, search may miss recent additions." A Principal-Engineer detail: contextd indexes newest-mtime-first so the files you most likely want appear before the backlog drains. The UI never pretends to certainty it doesn't have.

Drag-to-AI, on-device Local

Drop any file on the sidebar well to act on it — summarize, translate to Khmer, rename by content, find similar. Every action runs against the local LLM. The drop target is the same surface that shows "0 bytes left this machine": the affordance and the privacy promise occupy the same pixels.
