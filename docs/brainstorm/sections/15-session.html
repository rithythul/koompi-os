Full-screen scrim

Session overlay uses a dark semi-transparent scrim (rgba(7,11,19,0.88)) covering the entire screen. Click outside the grid dismisses. Esc also dismisses.

Keyboard navigation

Arrow keys navigate the grid. Enter selects. Tab wraps. Focused button gets a subtle white border highlight. Fully accessible without a mouse.

Danger confirmation for Shutdown/Reboot

Shutdown and Reboot show a secondary confirmation step: button text changes to "Confirm?" with a 3-second auto-cancel timer. Prevents accidental power-off.
