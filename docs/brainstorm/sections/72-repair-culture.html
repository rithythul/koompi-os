Below the device floor the OS sheds ambition, not function: memory capture and recall stay on, heavy consolidation batches into rarer, longer sleeps. Degrade local, never silent cloud (ADR-0001) — a weak machine gets a patient OS, not an upsell.

SMART data, ECC counts, and battery telemetry become plain-language foresight: "this SSD has about 8 months — your memory palace lives there, let's move it tonight." Failures become scheduled maintenance instead of lost decades.

Old machines earn honor, not shame

The machine biography — assembled 2026, third SSD, second screen, fourteen years of service — reframes age as proof of stewardship. In repair-stall economies this is reality; KOOMPI makes it identity.

Anti-obsolescence, made concrete

Planned obsolescence is a software decision wearing a hardware costume. By running gracefully below the floor and treating every swapped part as routine, the OS removes the manufactured reason to throw a working machine away.
