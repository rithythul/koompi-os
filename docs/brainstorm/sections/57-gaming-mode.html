Sovereignty includes off KOOMPI

Windows nags, GNOME hides the toggle, KDE buries it in Game Mode plugins. KOOMPI makes "turn the thinking off" a first-class, one-tap state — and means it. In Gaming Mode the local LLM does not infer, the indexer does not read your files, and no wake word listens. Stepping back is a feature, not a degraded mode.

Every change is a reversible delta

The daemon never "sets and forgets." Each tweak — governor, blur, EPP, notification hold — is journalled with its prior value. Exit replays the journal in reverse; a crash triggers a login-time watchdog restore. The aesthetic desktop always comes back, exactly as it was.

Honest about entry hardware

Built for Cambodia, where the common rig is a 4 GB laptop GPU, not a desktop 4090. The thermal panel warns before sustained throttle and coil whine, and offers a concrete fps cap — respecting the user's actual machine instead of assuming a flagship.

Ages with the library

Profiles key on stable Steam AppIDs and launcher YAML, not screen-scraped names — so a per-game tweak set in 2026 still auto-applies in 2046 after ten Proton versions and three GPUs. A hand-edited launch command locks to manual and is never silently overwritten across decades of updates.
