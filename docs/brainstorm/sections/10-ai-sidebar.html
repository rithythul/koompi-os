What's the current architecture for the bar?
The bar uses Quickshell + QML. Three styles: Hug (default), Taskbar (Win11), Dock (macOS). Bar content is in BarContent.qml with left/center/right zones. The center has Workspaces + Pomodoro. Right zone handles media, tray, brightness/volume scroll.

Local model first KOOMPI

Default mode runs a local LLM. Chip shows LOCAL in emerald. Switching to cloud shows amber CLOUD chip and triggers a consent prompt the first time.

RAG context panel KOOMPI

Shows exactly which files, docs, and memory entries the AI loaded. Full transparency — you always know why the AI said what it said.

Memory awareness KOOMPI

Persistent memories (user profile, project state, feedback) are visible and editable. AI uses them for personalization without hidden context.

Detach to window

AI sidebar can be popped out as a floating window (Super+A twice). Stays connected to the same session and context.
