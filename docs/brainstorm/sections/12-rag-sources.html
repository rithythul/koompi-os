Visual bar shows how much context window has been consumed. Helps you understand when to trim context or start a fresh conversation.

Source types

FILE = local filesystem, DOC = indexed document, MEM = persistent memory entry. Each type is color-coded: file=default, doc=accent, mem=cyan.

Edit context

Clicking any source row removes it from context. Drag to reorder by priority. Add new sources via the + button (opens file picker or memory browser).
